3 Ways of setting the width of a flex child:

  1. By default, the width will be the width of the element's content (see first flex example).
  2. Set a fixed width in pixels using the width property (see 200px sidebar example).
  3. Set a portion width using the flex property (see four column example).

* If you mix the width and the flex technique, than the flex portion is based on the remaining width after the fixed width is allocated (see 200px sidebar example).
